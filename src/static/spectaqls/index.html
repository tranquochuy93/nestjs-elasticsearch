Queries

billings

Response

Returns [BillingEntity!]!

Example

Query
query Billings {
  billings {
    id
    name
    totalOfPrice
    userId
    productBillings {
      amountOfProduct
      productId
      billingId
      product {
        id
        name
        price
      }
    }
  }
}
Response
{
  "data": {
    "billings": [
      {
        "id": "abc123",
        "name": "abc123",
        "totalOfPrice": 123.45,
        "userId": "abc123",
        "productBillings": [ProductBillingEntity]
      }
    ]
  }
}
